How to read: This report is a sequence of Benchmark.js runs. For each benchmark, the name of the benchmark is listed then a Benchmark.js run for each version of AngularJS is shown.

A Benchmark.js run consists of cycles. Each cycle runs the benchmarked function a pre-determined number of times in a tight loop. The number of cycles in the run is listed after run's name.

The run's chart plots time on the x-axis. The scale is constant across all runs in the report. The y-axis plots operations per second for each cycle.

The cycles with the max and min ops/sec are identified along with all the stats for that cycle.

The mean ops/sec over all the cycles is identified by a run line.
